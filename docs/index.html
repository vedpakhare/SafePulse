Installation

Get started in minutes

Kali Linux / Debian-based Systems

Terminal Commands
git clone https://github.com/vedpakhare/SafePulse.git
cd SafePulse
chmod +x install.sh
./install.sh

Other Linux Systems

Dependencies First
sudo apt update
sudo apt install python3 pip3 chkrootkit nmap ufw
Run SafePulse
# CLI Version
./src/safepulse.sh

# GUI Version
python3 src/gui.py
