Installation

Get started in minutes

Kali Linux / Debian-based Systems

Terminal Commands
git clone https://github.com/vedpakhare/SafePulse.git
cd SafePulse
chmod +x install.sh
./install.sh

Other Linux Systems

Dependencies First
sudo apt update
sudo apt install python3 pip3 chkrootkit nmap ufw
Run SafePulse
# CLI Version
./src/safepulse.sh

# GUI Version
python3 src/gui.py

Get Started in 60 Seconds

Works on Kali Linux, Ubuntu, and most Debian-based systems.

kali-terminal

Kali Linux / Debian

Run these commands:

# Clone repository
git clone https://github.com/vedpakhare/SafePulse.git
cd SafePulse

# Install
chmod +x install.sh
./install.sh

# Run CLI version
./src/safepulse.sh

# Or GUI version
python3 src/gui.py

Other Linux Systems

1. Install Dependencies

sudo apt update
sudo apt install python3 pip3 chkrootkit nmap ufw

2. Manual Installation

# Download manually from GitHub
# Extract and run:
python3 src/scanner.py

3. Run Scanner

# CLI version
./src/safepulse.sh

# GUI version
python3 src/gui.py
